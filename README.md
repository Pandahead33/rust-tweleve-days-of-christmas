# rust-tweleve-days-of-christmas
Generate the cumulative song The Twelve Days of Christmas using the Rust programming language.

# How to run
1. Install Rust (rustup is the easiest way). https://www.rust-lang.org/tools/install
2. Go into the folder and run:

```sh 
cargo run main
```

You should see the lyrics to the song printed out and the program will end!
